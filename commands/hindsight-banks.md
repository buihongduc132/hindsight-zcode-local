# /hindsight-banks

Show the Hindsight banks connected to this project.

## What to do

1. Run the `hindsight_banks` tool with `listAll: false` to show this project's resolved bank + global bank.
2. Present the bank IDs, fact counts, top tags, and top entities.
3. If the user wants to see every bank on the server, run again with `listAll: true`.
